导出功能同浏览页面查询，通过解析“标准页面-列表”中配置文件实现。

### 数据特殊处理

如果需要对查询出的结果进行加工，在BLL类中实现ExportBaseItemAfter方法，参数为查询结果的DataTable，返回处理后的DataTable。
操作同<font color=red>**标准页面-列表**</font>
